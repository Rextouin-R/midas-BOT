## Install 
```
pkg install git
```
```
pkg install nano
```
```
pkg install python
```
```
pkg install python3 python3-pip
```
```
git clone https://github.com/Rextouin-R/midas-BOT/
```
## Buka file 
```
cd midas-BOT
```
```
python -m pip install -r requirements.txt
```
## Edit `data.txt` lalu paste query_id/user_id kalian
## Save `CTRL+x-y` (enter) 
```
nano data.txt
```
## Run bot
```
python midas.py
```
